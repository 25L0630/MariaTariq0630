# GreenEnergy Solar Website

This project is a simple, responsive solar energy promotional website containing:
- A hero section  
- A features section  
- A consultation sign-up form  
- Interactive CSS components  
- A responsive layout (mobile + tablet support)

style.css
This file contains all the CSS styles, including:
- Colors & variables
- Layout (Grid + Flexbox)
- Hover effects
- Form styling
- Responsive design using media queries

Features Included
- Responsive design
- Image
- Expandable info box using CSS only
- Button hover animations
- Accessible focus states
- SVG solar icon in hero section
